Закон «О государственных кодах реагирования штата San Andreas»

Раздел 1. Общее положение
1.1 Настоящий Закон определяет систему государственных кодов и протоколов, используемых сотрудниками государственных, силовых и спасательных структур штата Сан-Андреас.

1.2 Закон носит информационно-справочный характер и направлен на обеспечение единообразного понимания и применения служебных сигналов, сообщений и кодов.





Статья 2. Цели и задачи
2.1 Целями настоящего Закона являются:
повышение эффективности межведомственного взаимодействия;
сокращение времени передачи и обработки информации в чрезвычайных ситуациях;
обеспечение точности и однозначности служебных сообщений.
2.2 Задачами Закона являются:
установление перечня кодов и протоколов, применяемых на территории штата;
определение правил их использования.


Раздел 3. Тен-коды
3.1. - Тен-коды используется для сокращения передачи важной информации.
3.2. - В случае возможности использования тен-кода для передачи информации, тен-код должен обладать приоритетом.
3.3. - Тен-кодами считаются:
10-0 — Отмена или отбой;
10-1 — Не принял сообщение, повторите;
10-3 — Прекратить трансляцию;
10-4 — Принято;
10-5 — Повторите последнее сообщение;
10-6 — Не принял/нет/отрицательно;
10-7 — Конец связи;
— Готов к работе;
— В «FIB» посетитель;
— Проникновение в «FIB»;
— Возвращаюсь в «FIB»;
— Локация или Местоположение;
— Направляюсь на локацию;
— Требуется эвакуация;
— Запрос информации;
— Провожу обыск задержанного;
— Выехал на поддержку или выезжаю на подкрепление;
— Требуется подкрепление/запрос поддержки;
— Требуется медицинская помощь;
— Провожу задержание;
— Ситуация урегулирована или отбой.

Раздел 4. Ситуативные коды
4.1. - Ситуативные коды используются для передачи важной информации, на планшете.
4.2. - Ситуативными кодами признаются:
CODE 0 - Офицер убит или тяжело ранен;
CODE 1 - Офицер ранен;
CODE 2 - Обычный вызов помощи;
CODE 3 - Срочный вызов;
CODE 4 - Не требуется помощь на ситуации или ситуация окончена;
CODE 5 - Ложная тревога;
- Массовые беспорядки;
- Требуется срочная огневая поддержка/массовое скопление криминальных элементов;
- Похищение или попытка похищения человека;
- Угроза охраняемому лицу.
