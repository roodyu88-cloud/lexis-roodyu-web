Положение об “Юрисдикции, кодах и протоколах”

Раздел 1. Общее положение
1.1. - Настоящее положение регулирует юрисдикции, специальные протоколы и тревоги, на территории штата Сан - Андреас.
1.2. - На любых специальных операциях, включая особые протоколы оперативное руководство над ситуацией возлагается на федеральное бюро расследований.

Раздел 2. Юрисдикция
2.1. - В случае открытие юрисдикции, действие региональных государственных департаментов, агентств, бюро и таких других структур не ограничивается юрисдикцией.
2.2. - Правом на открытие юрисдикции обладает Губернатор, Вице-губернатор, а также Министр Национальной Безопасности.
2.3. - Сотрудники специальных следственных отделов ( подразделений ), а также сотрудники находящиеся под “прикрытием”, региональных правоохранительных структур уполномочены осуществлять свою деятельность на территории всего штата.
2.4. - В случае военного или чрезвычайного положения юрисдикция считается открытой.
2.5. - В случае проведения специальных операций, сотрудники правоохранительных ведомств имеют право действовать на территории всего штата.
2.6. - Сотрудники “Военной Полиции” имеют право действовать на территории всего штата.
2.7. - В случае проведения воздушных патрулей, сотрудники национальной гвардии могут действовать на территории всего штата.

Раздел 3. Тен-коды
3.1. - Тен-коды используется для сокращения передачи важной информации.
3.2. - В случае возможности использования тен-кода для передачи информации, тен-код должен обладать приоритетом.
3.3. - Тен-кодами считаются:
10-0 — Отмена или отбой;
10-1 — Не принял сообщение, повторите;
10-3 — Прекратить трансляцию;
10-4 — Принято;
10-5 — Повторите последнее сообщение;
10-6 — Не принял/нет/отрицательно;
10-7 — Конец связи;
10-8 — Готов к работе;
10-12 — В «FIB» посетитель;
10-14 — Проникновение в «FIB»;
10-15 — Возвращаюсь в «FIB»;
10-20 — Локация или Местоположение;
10-21 — Направляюсь на локацию;
10-33 — Требуется эвакуация;
10-43 — Запрос информации;
10-46 — Провожу обыск задержанного;
10-50 — Выехал на поддержку или выезжаю на подкрепление;
10-70 — Требуется подкрепление/запрос поддержки;
10-71 — Требуется медицинская помощь;
10-75 — Провожу задержание;
10-99 — Ситуация урегулирована или отбой.

Раздел 4. Ситуативные коды
4.1. - Ситуативные коды используются для передачи важной информации, на планшете.
4.2. - Ситуативными кодами признаются:
CODE 0 - Офицер убит или тяжело ранен;
CODE 1 - Офицер ранен;
CODE 2 - Обычный вызов помощи;
CODE 3 - Срочный вызов;
CODE 4 - Не требуется помощь на ситуации или ситуация окончена;
CODE 5 - Ложная тревога;
CODE 6: Массовые беспорядки;
CODE 7 - Требуется срочная огневая поддержка/массовое скопление криминальных элементов;
CODE 8 - Похищение или попытка похищения человека;
CODE 9 - Угроза охраняемому лицу.

Раздел 5. Положение о протоколах
5.1. - Протоколы используется, для неотложного обозначения ситуации, и требуют срочной реакции на последние.
5.2. - Протоколами признаются:
Протокол “Код Красный”;
Протокол “Белый Дом”;
Протокол “Банковая Защита”;
Протокол “Захват”;
5.3. - Под протоколом “Код Красный” понимается нападение на форт-занкудо.
5.3.1. - В случае введения протокола “Код Красный” все свободные сотрудники ( юниты ) должны незамедлительно прибыть на территорию форта занкудо.
5.4. - Под протоколом “Белый Дом” понимается нападение на мэрию лос-сантос.
5.4.1. - В случае введения протокола “Белый Дом” все свободные сотрудники ( юниты ) должны незамедлительно прибыть на территорию мэрии лос-сантос.
5.5. Под протоколом “Банковая Защита” понимается нападение на банки.
5.5.1. - В случае введения протокола “Банковая Защита” все свободные сотрудники ( юниты ) должны незамедлительно прибыть на территорию банка, подверженного нападению.
5.6. - В случае нападения на территории государственных структур, исключая мэрию и форт-занкудо вводится протокол “Захват”.
5.6.1. - В случае введения протокола “Захват” все свободные сотрудники ( юниты ) должны незамедлительно прибыть на территорию учреждения, подверженного нападению.
