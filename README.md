# Lexis Web (main / placeholder)

Веб-платформа Lexis: пресеты, ИИ-ассистент по законам RP-серверов, тренажёр экзаменов,
релизы и админ-панель. Эта ветка (`main`) — витрина с плейсхолдер-данными и клиентской
dev-авторизацией (статический экспорт), для локального дизайн-QA. В прод не разворачивается.

> Реальная версия с БД (PostgreSQL/Prisma) и Discord-авторизацией — ветка `prod-port`
> (репозиторий `roodyu-web-prod`). Именно она разворачивается на **lexis.xyz**, вместе с
> гайдом по деплою `DEPLOYMENT.md`.

## Что менялось по ходу проекта

Краткая история проекта от первого коммита до текущего состояния — в [CHANGELOG.md](./CHANGELOG.md).

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000). Роль пользователя переключается
через dev-переключатель ролей в UI (`DevAuthProvider`) — реальная авторизация в этой
ветке не задействована.
