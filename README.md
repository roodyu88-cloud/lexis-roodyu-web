# Lexis Web (prod-port)

Веб-платформа Lexis: пресеты, ИИ-ассистент по законам RP-серверов, тренажёр экзаменов,
релизы и админ-панель. Эта ветка (`prod-port`) — рабочая версия с реальной БД (PostgreSQL/Prisma)
и реальной Discord-авторизацией. Именно она разворачивается в прод на **lexis.xyz**.

> Ветка `main` этого проекта (репозиторий `roodyu-web`) — витрина с плейсхолдер-данными
> для локального дизайн-QA, в прод не разворачивается.

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000). Для локального запуска нужны
переменные окружения из `DATABASE_URL`, `NEXTAUTH_SECRET`, `DISCORD_CLIENT_ID`/`DISCORD_CLIENT_SECRET`
и хотя бы один ключ ИИ-провайдера (см. [DEPLOYMENT.md](./DEPLOYMENT.md#2-переменные-окружения-vercel--project-settings--environment-variables)).
