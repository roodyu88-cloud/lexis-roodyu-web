# Что менялось по ходу проекта (кратко)

Хронология проекта от первого коммита до текущего состояния. Гайд по деплою (`DEPLOYMENT.md`) лежит в ветке `prod-port` — именно она разворачивается в прод, эта ветка (`main`) плейсхолдер-витрина для дизайн-QA.

## Фундамент
- Next.js-приложение с нуля: Prisma-схема, страницы пресетов/релизов/экзамена/ассистента, база законов серверов (`majesticrp/`, `crystalrp/`, `revengerp/`).
- БД переведена с SQLite на PostgreSQL.
- Добавлена авторизация через Discord (NextAuth), роли пользователей (admin/developer/user), бан-система (в т.ч. по HWID).

## ИИ-ассистент
- Роут `/api/assistant` дорабатывался итеративно: фолбэки между Gemini → DeepSeek → OpenRouter, увеличение лимита системного промпта, исправление сопоставления файлов законов, логирование запросов, генерация пресетов из ответа ИИ (с автоочисткой через 30 дней).

## Функциональность
- Пресеты: создание/редактирование/верификация, фильтрация по серверам.
- Релизы: загрузка, редактирование, вебхуки, скачивание.
- Тренажёр экзаменов (`/exam`): генерация, прохождение, история результатов.
- Промокоды и премиум-подписка (активация через Discord-тикет).
- Админ-панель: управление пользователями, серверами, промокодами, релизами, новостями, статистикой.

## Разделение на две ветки
- **`main`** → репозиторий `roodyu-web`: витрина с плейсхолдер-данными (client-side dev-auth, статический экспорт) для локального дизайн-QA. На прод не разворачивается.
- **`prod-port`** → репозиторий `roodyu-web-prod`: реальная БД (Postgres/Prisma) и реальная Discord-авторизация. Именно эта ветка идёт в прод (lexis.xyz).

## Дизайн-редизайны (перенесены с `main` на `prod-port` без изменения auth/admin-логики)
1. **Raycast-редизайн** — переход на тёмную палитру (`--color-void-black`), сине-фиолетовый акцент вместо кораллового, компоненты `rc-card-edge`/`rc-btn`.
2. **Void-referenced структурная переработка** — sticky-pin hero, объединённый аккордеон преимуществ, рескейл страницы экзамена, glassmorphism-админка.
3. **Round 2** — устранение шва под шапкой на hero, редизайн ghost-кнопок, FAQ в виде аккордеона, выравнивание иконок навбара.
4. **Round 3** — pin-scroll hero, промо-полоса, ребрендинг PRO → PREMIUM, вижард тренажёра, редизайн админки.
5. **Round 4** — уборка hero, promo → закреплённое уведомление, редизайн `/premium`, `/guide`, `/releases`.
6. **Round 5** — фикс клика по уведомлению, устранение дубля промо-баннера в футере, фикс мёртвой ссылки в hero, унификация стиля `/promo`.
